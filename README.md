# Missing Child Identification System

## Description
This project focuses on identifying missing children using image-based recognition.
It compares uploaded images with stored child images to help authorities in identification.

## Technologies Used
- Node.js
- Express.js
- Image Processing
- HTML, CSS, JavaScript

## Features
- Upload child image
- Compare with database
- Real-time identification
- Secure backend

## How to Run
1. Install Node.js
2. Open terminal in backend folder
3. Run:
   npm install
   node server.js
